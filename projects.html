KubeFlex

A carbon-aware Kubernetes scheduler that optimizes workload placement based on renewable energy availability, featuring live migration capabilities.

Key Features

  • Carbon-Aware Scheduling: Intelligently places workloads on nodes with access to renewable energy
  • Live Migration: Seamlessly migrates running containers without downtime
  • Energy Optimization: Reduces carbon footprint by prioritizing green energy sources
  • Kubernetes Integration: Built as a custom scheduler for Kubernetes clusters

Impact

This project addresses the growing need for sustainable computing by enabling cloud infrastructure to automatically optimize for carbon emissions. By aligning workload placement with renewable energy availability, KubeFlex helps organizations reduce their environmental impact while maintaining system performance.

Locker

An oblivious datastore system designed to protect data access patterns from potential adversaries through advanced cryptographic techniques.

Key Features

  • Oblivious Access Patterns: Hides data access patterns from observers
  • Cryptographic Security: Uses state-of-the-art cryptographic primitives
  • Performance Optimization: Balances security with practical performance
  • Privacy Protection: Ensures data privacy even under adversarial conditions

Impact

In an era where data privacy is paramount, Locker provides a crucial layer of protection for sensitive data access patterns. This is particularly important for systems handling confidential information where access patterns themselves could reveal sensitive information.
