Capstone Project

KubeFlex: A Carbon-Aware Scheduler with Live Migration

View on GitHub

Synopsis

KubeFlex is a carbon-aware Kubernetes scheduler designed to optimize workload placement based on renewable energy availability. The system features live migration capabilities, allowing workloads to be dynamically moved to nodes powered by renewable energy sources when available. This project addresses the growing need for sustainable computing by enabling cloud infrastructure to automatically optimize for carbon emissions, helping organizations reduce their environmental impact while maintaining system performance.

Keywords

Kubernetes Carbon-Aware Computing Live Migration Green Computing Container Orchestration Renewable Energy Workload Scheduling

SADE: Safety Aligned Dataset Engine

View on GitHub

Synopsis

SADE (Safety Aligned Dataset Engine) is a dataset engine focused on safety alignment, ensuring AI models are trained on datasets that promote safe and ethical behavior. The system helps create training datasets that align with safety and ethical guidelines. As AI systems become more prevalent, ensuring they behave safely and ethically is crucial. SADE addresses this need by providing tools to create training datasets that inherently promote safe and ethical AI behavior, contributing to the development of more trustworthy AI systems.

Keywords

AI Safety Dataset Engineering Ethical AI Machine Learning Data Quality AI Alignment

Locker: An Oblivious Datastore for Protecting Data Access Patterns

View on GitHub

Synopsis

Locker is an oblivious datastore system that protects data access patterns from potential adversaries. The system uses advanced cryptographic techniques to ensure that even an adversary with full visibility into the system's operations cannot determine which data is being accessed. In an era where data privacy is paramount, Locker provides a crucial layer of protection for sensitive data access patterns, particularly important for systems handling confidential information where access patterns themselves could reveal sensitive information.

Keywords

Oblivious RAM Cryptography Data Privacy Access Pattern Protection Security Privacy-Preserving Systems

Speedly: Aggregating Traffic Analysis through Stream Processing

View on GitHub

Synopsis

Speedly is a high-performance stream processing system designed for real-time traffic analysis and aggregation. The system enables efficient network monitoring and analytics by processing data streams in real-time with minimal latency. In an increasingly connected world, the ability to analyze network traffic in real-time is essential for network security, performance optimization, and understanding network behavior. Speedly provides a robust foundation for building real-time network analytics systems.

Keywords

Stream Processing Real-Time Analytics Network Monitoring Traffic Analysis High Performance Computing Data Streaming
