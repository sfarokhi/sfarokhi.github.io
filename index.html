KubeFlex: A Carbon-Aware Scheduler with Live Migration

View on GitHub

Synopsis

KubeFlex is a carbon-aware Kubernetes scheduler I architected that optimizes workload placement based on renewable energy availability. I built a Kubernetes-native scheduler with carbon-aware workload orchestration and implemented live migration for dynamic placement optimization across distributed regions. The results were pretty impressive - we achieved 95-99% emission efficiency for day-long workloads, and migrations only took about 2 seconds. It was really rewarding to work on something that directly addresses sustainable computing and helps reduce the environmental impact of cloud infrastructure.
